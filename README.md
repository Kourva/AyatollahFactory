# Install
+ clone
```bash
git clone https://github.com/Kourva/AyatollahFactory && cd AyatollahFactory && chmod +x install.sh
```
+ requirements
```bash
./install.sh
```
+ run GUI
```bash
python3 main.py
```
+ Run CLI
```bash
python3 cli.py
```
